# 🧠 PRODUCT ONE-PAGER

## MyWally — Wallet Companion for Confident Financial

## Participation

# 📌 EXECUTIVE SUMMARY

Malaysia’s elderly population is growing — but while many already have access to digital
wallets, **they are not confidently using them**.
The core issue is not access, but **fear and lack of confidence** :
● Fear of making mistakes
● Uncertainty in understanding finances
● Lack of real-time support when making decisions
**MyWally** is a **confidence layer embedded within the existing eWallet** , designed to help
elderly users:
● Understand their money
● Make decisions safely
● Take action with support
👉 Instead of adding complexity, we **simplify the experience and introduce real-time human
support at critical moments**

# ❓ HOW MIGHT WE

```
How might we enable elderly users to confidently participate in digital
financial services using tools they already have — without increasing
complexity or risk?
```

# 👤 PERSONA & PROBLEM

## 🎯 Primary Persona: Makcik Aisyah (60–75)

## Profile

```
● Basic smartphone user (WhatsApp, simple apps)
● Limited or irregular income
● Relies partly on children
● Risk-averse
```

## Core Problems

```
● ❌ Doesn’t know if she has enough money daily
● ❌ Finds financial apps overwhelming
● ❌ Afraid of making wrong decisions
● ❌ No one to consult at the moment of transaction
● ❌ Avoids using digital financial tools
```

# 🧭 DESIGN PRINCIPLES

1. **Clarity over complexity**
   → Show only what matters now
2. **Guidance over exploration**
   → Users ask, not navigate
3. **Confidence over control**
   → Reduce fear before enabling action
4. **Support over isolation**
   → Decisions are not made alone
5. **Calm over alarm**
   → Avoid fear-based messaging

# 💡 SOLUTION (ONE-LINER)

```
MyWally is a wallet companion that enables elderly users to confidently
participate in digital finance — with real-time support from trusted family
when they need it
```

# 🧩 MODULES & FEATURES (MVP)

## 🟦 1. Elderly Mode (Enable)

**Purpose:** Reduce cognitive load instantly
**Features:**
● Simplified UI inside existing eWallet
● Large buttons, minimal actions
● Clear balance & daily summary

## 🟨 2. Wally (Engage)

**Purpose:** Help users understand and decide
**Features:**
● Voice / button-based questions
● Simple answers:
○ “Do I have enough?”
○ “How much did I spend?”
● Human-readable insights

## 🟧 3. Daily Budget & Feedback

**Purpose:** Build financial awareness
**Features:**
● Simple budget setup

```
● Daily spending feedback
● Status indicators:
○ Safe / Near / Exceeded
```

## 🟥 4. Real-Time Support Layer (Empower) — ⭐ KEY

## DIFFERENTIATOR

**Purpose:** Enable safe decision-making at critical moments
**Features:**
● Detect unusual transaction behaviour
● Trigger **AI call to caregiver**
● Caregiver can:
○ Approve
○ Block
● Elderly receives:
○ Calm explanation
○ Clear next step

## 🟩 5. Trust & Guidance Layer

**Purpose:** Build long-term confidence
**Features:**
● Gentle financial tips
● Reassurance messaging
● No blame / no fear tone

# 🎯 MVP FOCUS (HACKATHON)

**Must Have:**
● Elderly Mode UI
● Wally (basic Q&A)

● Transaction flow + caregiver approval simulation
● Blocked / approved UX states
**Nice to Have:**
● Voice interaction
● Budget tracking
● Safety tips

# 🧠 FINAL PRODUCT INTENT

```
We are not building a financial tool —
we are building a confidence layer on top of finance
```
